## Track 2: Fine-Grained Aesthetic Dimension Prediction

For Track 2, we have included a number of special samples designed to test model sensitivity to individual aesthetic dimensions. Each of these samples exhibits degradation in only one or two specific dimensions rather than an overall aesthetic drop, encouraging more fine-grained prediction capability.
