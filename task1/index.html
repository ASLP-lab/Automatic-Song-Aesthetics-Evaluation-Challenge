Task1s

Description Participants must predict a single holistic aesthetic score for each song, representing an overall musical impression of the song’s artistic quality (corresponding to the Overall Musicality in SongEval dataset). Evaluation will be based on subjective scores from a panel of musician raters.

Test Set Two types of test sets are provided for Track 1: Set 1(Regular Set): Contains songs generated by the same models used in the training set. Set 2(Hard Set): Includes songs generated by 10 different open-source models, offering a more diverse and challenging evaluation scenario.
