Test Set

Two types of test sets are provided for Track 1:

Set 1(Regular Set): Contains songs generated by the same models used in the training set.

Set 2(Hard Set): Includes songs generated by 10 different open-source models, offering a more diverse and challenging evaluation scenario. This set aims to assess each model’s generalization ability to unseen generation systems.
